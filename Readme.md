# Using Ontologies for Adaptive Planning and Robust Execution in Robotic Manipulation


## Description

This repository contains the knowledge files supporting the implementation and evaluation of the methods presented in the paper "Using Ontologies for Adaptive Planning and Robust Execution in Robotic Manipulation". These files correspond to the work presented to on the **Planning and Ontology Workshop (PLATO’25), September 9, 2025, Catania, Italy**

## MRS Structures

This section illustrates the Monitoring and Recovery Structures (MRS) presented in the work. 

The implemented MRS are:
- Precondition
- Holdcondition
- Postcondition
- Action Recovery
- Retry

The data in this repository uses generic behaviors (Behavior A, Behavior B, Condition I...) to illustrate the functioning. Dummy flavors have been implemented, to show the application of the MRS structures automatically when generating Behavior Trees with ontological knowledge support.
